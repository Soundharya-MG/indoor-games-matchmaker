# 🛡️ SportsBuddy Finder 🏆

SportsBuddy is a real-time web platform that helps users find partners for sports and indoor games based on their current location. No more waiting for a team; find your match and start playing!

## ✨ Key Features
* **📍 Location-Based Discovery:** Uses Geolocation API to find players within your vicinity.
* **🏸 Multi-Sport Support:** Support for Badminton, Chess, Table Tennis, Cricket, and more.
* **📱 Responsive Dashboard:** A glassmorphism-themed professional UI for all devices.
* **⚡ Real-time Updates:** Instant feedback on available matches and play zones.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+)
* **Backend:** Node.js, Express.js
* **Mapping:** Leaflet.js API
* **Database:** MongoDB (or Local JSON Storage for POC)
